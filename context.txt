We are building an interactive AI drawing recognition web app for the El Paso Science Festival. The app is being hosted at a ColorStack booth and must be UTEP-themed. What the app does:
A visitor draws a simple doodle on an HTML5 canvas. They hit a button and the app sends the canvas image to the Claude API (claude-sonnet-4-20250514) as a base64-encoded PNG. Claude identifies what was drawn, returns the top 3 guesses with confidence scores, and generates a UTEP college/department connection with a short explanation and a fun fact — all in a single API call. The result is displayed in a card on the right side of the screen.
Current architecture:

Single HTML file, vanilla JS, no dependencies except Google Fonts
Canvas drawing with pointer events (supports mouse, touch, and pressure-sensitive stylus via e.pressure)
On "What Did I Draw?" button click: canvas.toDataURL() → base64 PNG → POST to https://api.anthropic.com/v1/messages
Model: claude-sonnet-4-20250514, max_tokens: 600

  }
}

Confetti animation fires on each successful result
